# ECHIPA_04_PIU_2025
Dezvoltarea unei soluții de management financiar pentru familii fără studii de specialitate. Obiectivul cheie este monitorizarea detaliată (eroare de +/- 10 RON pe lună) a veniturilor, economiilor și cheltuielilor. Proiectul se axează pe educația financiară a copiilor și pe implicarea lor în gestionarea resurselor.
